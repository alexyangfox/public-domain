**********************
API and ABI Versioning
**********************

``libcpr`` uses a three-level "X.Y.Z" API versioning scheme, as follows:

* Patch versions (e.g., 0.1.**3**, 1.0.**1**, 1.2.**8**).

* Minor versions (e.g., 0.**4**.0, 1.**1**.0).

* Major versions.
